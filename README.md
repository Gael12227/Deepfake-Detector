# Deepfake Detector

A machine learning project designed to detect AI-generated images. This repository contains the inference scripts and the trained model weights.

## 🚀 Project Overview
- **App:** `app.py` (Main entry point)
- **Model:** `funet_a_full.pth` (Pre-trained CNN and GNN)
- **Architecture:** Defined in `model_definitions.py` and `my_models.py`
